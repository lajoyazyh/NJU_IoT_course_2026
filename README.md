## 实验一：智能家居网络模拟

**目标**：通过 Cisco Packet Tracer 搭建虚拟智能家居网络，理解物联网端-边-云架构与条件联动规则。

**实验内容**：
1. **创建智能家居网络**：配置无线路由器、IoT Server，注册智能台灯与摄像头，通过平板/手机远程控制。
2. **模拟防盗系统**：部署绊线传感器、智能门锁、监控摄像头，编写条件规则实现"哨兵模式"——传感器触发 → 门锁反锁 + 摄像头开启。
3. **模拟智能恒温养殖场**（选做）：温度 20°C~30°C 区间自动调温，摄像头开启联动灯光照明。

## 实验二：人脸表情识别

**目标**：训练 CNN 深度学习模型识别七种人脸表情（Happy、Neutral、Sad、Angry、Surprise、Fear、Disgust），掌握视觉分类全流程。

**技术栈**：
- **模型**：自建 CNN（3 层卷积 + 池化 + 全连接），最终验证准确率约 63%。
- **框架**：TensorFlow / Keras，支持 GPU 加速训练。
- **数据增广**：随机翻转、旋转、平移、缩放等增强策略。
- **推理**：支持单张图片、批量图片、摄像头实时识别。

## 实验三：手势识别

**目标**：基于 MediaPipe 的 21 点手部关键点检测 + TFLite 分类模型，实现静态手势识别（Open、Close、Pointer、OK）和动态手势识别（Stop、Clockwise、Counter Clockwise、Move）。

**技术栈**：
- **手部检测**：MediaPipe Hands，提取 21 个手部关键点坐标。
- **手势分类**：TensorFlow Lite 模型，输入归一化关键点特征。
- **动态识别**：基于食指指尖历史轨迹的序列分类。
- **模式**：支持图片模式和摄像头实时模式。

## 大作业：课堂状态分析系统

**目标**：综合实验二的人脸表情识别能力，构建一个完整的课堂状态分析 Web 应用，支持图片、视频、摄像头三种输入模式。

**核心特性**：
- **多人脸检测**：MediaPipe + Haar Cascade **双检测器合并**，NMS 去重，大幅提升侧脸/模糊人脸的召回率。
- **表情识别**：复用实验二的 CNN 模型权重，支持快速模式（跳过增广，提速 ~6×）和高精度模式。
- **统计分析**：表情分布柱状图、课堂专注度评级、历史趋势折线图。
- **记录管理**：自动保存 CSV 记录，支持导出与历史回看。
- **WebRTC 实时流**：摄像头实时人脸检测与表情标注。

**技术栈**：Streamlit · OpenCV · MediaPipe · TensorFlow/Keras · Pandas · streamlit-webrtc
